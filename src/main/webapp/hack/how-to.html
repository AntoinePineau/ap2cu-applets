How to use Image Hack ?

Aim

send a mail to a receiver with the image hack, you will receive an acknowledgement mail with the location of the receiver at the time he opens the mail

URL

The service is available at the following URL: http://ap2cu.com/tools/hack

Parameters

Three parameters are available:
  • mailto: your mail address where you want to receive the acknowledgement mail

  • img: the image to display (full URL).
    http://ap2cu.com/tools/icomatics/emoticons/emotes/16/happy.png is the default value.

  • subject: the subject of the acknowledgement mail (URL encoded).
    IP lookup hack is the default value.



Example

http://ap2cu.com/tools/hack/?mailto=antoine.pineau@gmail.com&subject=Image%20Hack
