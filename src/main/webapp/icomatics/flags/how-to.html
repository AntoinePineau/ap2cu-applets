How to use Flag icomatics ?

URL

The service is available at the following URL: http://ap2cu.com/tools/icomatics/flags

Parameters

Three parameters are available:
  • country: the ISO-3166-1 alpha-2 country code, on 2 characters.
    __ is the default value, meaning "unknown country".

  • size: the size of the flag.
    3 values are available: small, normal, big.
    normal is the default value.

  • type: the format of the image.
    3 values are available: PNG, GIF and JPEG.
    PNG is the default value.

Examples

  1. Big French Flag in JPEG:
        <img src="http://ap2cu.com/tools/icomatics/flags/?country=FR&size=big&type=JPEG" alt="France" />

  2. Normal US Flag in GIF:
        <img src="http://ap2cu.com/tools/icomatics/flags/?country=US&type=GIF" alt="US" />

  3. Small Spain Flag in PNG:
        <img src="http://ap2cu.com/tools/icomatics/flags/?country=ES&size=small" alt="Spain" />

  4. Default Flag:
        <img src="http://ap2cu.com/tools/icomatics/flags" alt="Default" />
